Tamanho da fontes

Exitem dois tipos de medidas para definir o tamanho da fonte:

Medidas Absolutas:
cm, mm, in, px, pt, pc ...

Medidas Relativas:
em, ex, rem, vw, vh, %, ...

Os editores de textos utilizam PT na configuração do tamanho das letras.
16pt = 1em
24pt = 1.5em
32pt = 2em

Usando estilos em fontes

Utiliza-se as configurações detro do 'head - style':
font-family: 'work Sans', sans-serif;
font-weigth: bolder; = 'lighter' 'normal' 'bold' 'bolder'
font-size: 3em
font-style: italic

Voce pode abreviar usando a font.
" font: ilaic bolder 3em 'work Sans', san-serif; "
precisa esta nesta ordem:
- font-style -> font-weight -> font-size -> font-family
