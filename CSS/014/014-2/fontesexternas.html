Fontes google fonts

Voce entra no site fonts google e neste site, você encontrará vários tipos de fontes da internet.
Basta colocar em style a configuração import. 

    style
        @import url('https://fonts.googleapis.com/css2?
        family=Pacifico&display=swap');
        
        @import url('https://fonts.googleapis.com/css2?
        family=Pacifico&family=Work+Sans:ital,wght@0,100;0,200;0,
        300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,
        300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap');
    /style

Este endereço da 'url' voce copia no site font google.
Mas quando voce usar uma fonte externa use senpre junto fontes basicas para nao travar o site.
ex:
font-family: 'Pacifico', Arial, Helvetica, sans-serif;
font-family: 'Work Sans', Arial, Helvetica, sans-serif;
