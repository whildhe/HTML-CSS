Font Baixada.

Baixe uma fonte especifica e coloque numa pasta "fonts" junto com o index.html para organizar melhor. 

    style
        @font-face {
            font-family: 'scrito1';
            src: url('fonts/MeieScript-Regular.ttf') format('truetype');
            font-family: 'scrito2';
            src: url('fonts/MonsieurLaDoulaise-Regular.ttf') format('truetype');
        }
    /style

- Existe tipos de format:
- opentype (arquivo OTF)
- truetype (arquivo TTF)
- embedded-opentype
- truetype-aat (Apple Advanced Typography)
- svg
