Aprendendo Id

Id unico

O id serve para identificar um unico item não podendo ser usado nos demais.

h1 id="principal">Criando sites com HTML e CSS/h1
h1 id="segundo">Aprendendo ID /h1
h1 id="terceiro">Aprendendo CLASS /h1 .

Usando ID em stilo

Dentro do arquivo style voce coloca #nome

h1#principal {
text-align: center;
background-color: blue;
color: white;
}

Aprendendo Class

class multi

O uso do class é para ser usado em varias atigos:

h1 class="classe1">Aprendendo Id /h1
h1 class="classe1">Aprendendo class /h1
h2 class="classe2">Id unico /h2
h2 class="classe2">class multi /h2
 Para diciopnar uma class no meio de um paragrafo utiliso a TGA SPAN.
span class="classe3 "> h1 class="classe1">Aprendendo Id /h1

Usando class em stilo

Dentro do arquivo style voce coloca .nome

.classe1{
background-color: rgb(85, 85, 243);
.classe2{
background-color: rgb(182, 100, 62);
}
